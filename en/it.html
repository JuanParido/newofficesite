IT: Frequently Asked Questions

How can I connect my computer at the new site?

Preferably, we should use the WiFi wireless network NOSI. All Nokia and fALU computers have access granted to this network with no authentication required, since they have a digital certificate installed.

Internet access panel of Windows

Can I use a network cable to connect my PC?

Yes, you can, however it may not mean a performance improvement as well as there aren't enough network spots for everyone.

What is the BYOD network?

BYOD stands for Bring your own device. This network is set to connect personal devices and requires authentication by using your NSN-INTRA credentials. If you don't have these credentials, please contact Help Desk, since these will be used in many other services.

And the GUEST network?

This network is for visitors, customers and providers. A password must be requested at reception desk.s

How to print files

fALU mates can directly print, just after setting up their computers by following this guide:

  • Control Panel
  • Printers
  • HP Universal Printing PCL 6
  • Enter printer address (look for IP address in the printer tag, preferably in 10.xxx.xx.xx format)

We will have the follow-me function shortly, which is already used by fNokia mates.

I no longer have landline (PABX), how will I receive calls?

Nokia's standard is to use Jabber. You can only receive external calls if you have your mobile phone registered in the corporate directory. To upgrade, click on this link: https://uim.app.alcatel-lucent.com/

To use Jabber it is recommended to use a USB headphone - available in the catalog. But most headphones that come with mobile phones should work with Jabber.

How may I call to Help Desk since 2222 doesn't work from Jabber?

You may dial 8200700 or +1 630 224 5212
